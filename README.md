# Manufacturing-Performance-Dashboard-PowerBI
Manufacturing Performance Dashboard using Power BI

📌 Project Overview

This project presents an interactive Manufacturing Performance Dashboard developed in Power BI to monitor production, operational efficiency, manufacturing costs, and quality performance. The dashboard enables users to explore manufacturing data through interactive visualizations and KPIs, supporting better operational decision-making.


🎯 Project Objectives

Analyze manufacturing production performance.

Monitor operational efficiency using key performance indicators (KPIs).

Compare labour cost and energy consumption across production shifts.

Evaluate manufacturing quality using defect rate analysis.

Provide an interactive dashboard for business insights


🛠️ Tools & Technologies

Power BI Desktop

Power Query (Data Cleaning & Transformation)

DAX (Data Analysis Expressions)

Microsoft Excel (Initial data preparation)


📊 Key Performance Indicators (KPIs)

Total Units Produced

Total Production Volume (m³)

Units per Hour

Total Production Hours

Total Manufacturing Cost

Cost per Unit

Total Downtime Hours


📈 Dashboard Features

Production Trend Analysis

Production by Product Type

Labour Cost by Shift

Energy Consumption by Shift

Defect Rate Performance Gauge

Interactive Product Type and Shift slicers
